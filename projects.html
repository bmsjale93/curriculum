Web Análisis Subastas

Diseño Web - Scrapping Web

HTML - CSS - JAVASCRIPT - PHP - PYTHON

El proyecto incluye un script en Python que realiza web scraping para recopilar subastas, las cuales se limpian y almacenan en un CSV.

La web de subastas tiene acceso limitado y muestra las subastas en formato de tarjeta. Cada subasta incluye detalles como dirección, catastro, pujas, un mapa interactivo y una galería de imágenes. Además, permite subir subastas de manera individual o masiva, así como editarlas.

Simulación Phishing

Diseño Web - Ciberseguridad

HTML - CSS - JAVASCRIPT - PHP

4Phish es una aplicación web que permite a los administradores de empresas crear campañas de phishing para evaluar la seguridad de sus empleados.

Los administradores pueden ver métricas en tiempo real, como los empleados que han hecho clic en enlaces maliciosos y la cantidad de correos enviados. Si un empleado cae en la trampa, se le redirige a una página educativa sobre el phishing y puede descargar un PDF informativo para concienciarse sobre este tipo de estafa.

Balanpy - Cuida tu Mascota

Diseño Web

REACT JS

Balanpy es una aplicación web para gestionar el cuidado de mascotas, permitiendo a los usuarios organizar vacunas, citas médicas y alimentación desde un panel inspirado en juegos como Tamagotchi.

Los usuarios pueden visualizar estadísticas y gráficos sobre la salud de su mascota y reciben notificaciones sobre fechas importantes, como vacunas y desparasitaciones.

Portal de Empleo

Diseño Web

HTML - CSS - JAVASCRIPT - PHP

El Portal de Empleo es un proyecto completo que abarca diseño web, base de datos e interactividad.

Incluye un panel de control que permite administrar empleos, editar datos y subir currículums, brindando todas las funciones necesarias para un portal de empleo web desde su creación.
